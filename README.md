# python-一言

#### 介绍
灵感来自安卓端的一言
让终端充满诗意

#### 安装教程

复制到终端执行
deb系系统：
```
apt install python3 &&
apt install git &&
git clone https://gitee.com/Yinkelop/python---one-word.git &&
python3 .yiyan.py
```
arch:
```
pacman -S python3 &&
pacman -S git &&
git clone https://gitee.com/Yinkelop/python---one-word.git &&
python3 .yiyan.py
```
#### 使用说明

输入python3 .yiyan.py -h 获取帮助
